== Description ==

Merk Dir mit narando Online- und Printartikel, für die Du gerade keine Zeit hast – und lass sie Dir unterwegs vorlesen.

== Installation ==

= From your WordPress dashboard =

1. Visit 'Plugins > Add New'
2. Search for 'narando'
3. Activate narando from your Plugins page.
4. Visit 'Plugins > narando' and customize your player


= From WordPress.org =

1. Download narando.
2. Upload the 'narando' directory to your '/wp-content/plugins/' directory, using your favorite method (ftp, sftp, scp, etc...)
3. Activate narando from your Plugins page.
4. Visit 'Plugins > narando' and customize your player
